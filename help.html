Using AppTidy

AppTidy is designed around one safe habit: review the related files first, then move selected items to Trash.

Last updated: June 20, 2026

Quick Start

  • Open AppTidy and drop an app onto the drop area, or choose from the installed app list.
  • Review the scan results. Recommended items are selected by default.
  • Use All only when you recognize the review items and want a broader cleanup.
  • Click Remove. AppTidy moves files to Trash, not permanent deletion.

Restore Files

If you need to undo a cleanup, open Finder Trash and move items back to their original locations. AppTidy history can help you review what moved.

Full Disk Access

Full Disk Access helps AppTidy scan protected user data locations. macOS still requires you to approve access in System Settings. If AppTidy is not listed, click the plus button in Full Disk Access and choose AppTidy.app.

Review Items

Review means AppTidy found a possible leftover that may be shared, fuzzy-matched, or not uniquely owned by the selected app. These items are not automatically selected. Select them only when you recognize them.

Leftovers

The Leftovers screen finds orphaned caches and support folders that no installed app clearly owns. It is intentionally review-only because many developer tools and shared frameworks use generic folder names.

Administrator Approval

Apps installed in /Applications can be owned by root or an administrator account. When macOS requires approval, AppTidy asks through the native macOS password or Touch ID prompt and still moves items to Trash.

Smart Delete

Smart Delete watches for apps moved to Trash and can open AppTidy to review leftovers. Nothing is removed automatically.

Enterprise Deployment

Admins can deploy AppTidy with Microsoft Intune / Company Portal or Jamf Pro / Self Service or Self Service+. The enterprise guides include bundle ID, install path, macOS requirements, and optional PPPC notes.
