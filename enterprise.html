Deploy AppTidy with Intune or Jamf.

AppTidy can be distributed to managed Macs through Microsoft Intune / Company Portal or Jamf Pro / Self Service or Self Service+ using the signed and notarized direct-download package.

Last updated: June 20, 2026

Intune guide Jamf guide

Deployment Facts

Bundle ID com.apptidy.AppTidy
Install path /Applications/AppTidy.app
Minimum macOS macOS 13 or later
Distribution Signed and notarized DMG from app-tidy.com
Privacy permissions Full Disk Access helps AppTidy scan protected user data. It does not bypass administrator approval for root-owned apps in /Applications.

Recommended Model

For broad rollout, publish AppTidy as available in Company Portal or Jamf Self Service or Self Service+ first, then move to required deployment after a pilot. This gives admins time to verify signing, notarization, privacy prompts, and cleanup behavior on managed Macs.

Company Portal Jamf Self Service+ Signed DMG macOS 13+

About Full Disk Access

Organizations can manage privacy permissions with a PPPC configuration profile when they know the final signed app identity. Do not build PPPC payloads from ad-hoc debug builds because the code requirement changes. Full Disk Access is useful for scanning protected user data, but administrator-owned app bundles may still require native macOS approval.
